### Installation

1. Clone the repo

```sh
git clone git@github.com:brunogcpinheiro/klever-academy.git
```

2. Access project folder

```sh
cd klever-academy
```

3. Install dependencies

```sh
yarn install
npm install
```

4.  Run the project

```sh
yarn dev
npm run dev
```
